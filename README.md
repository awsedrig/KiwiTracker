#  KiwiTracker API

A goal tracking web application backend built with ASP.NET Core 8.0, featuring JWT authentication, clean architecture, and PostgreSQL database.

##  Tech Stack

- **Framework:** ASP.NET Core 8.0 Web API
- **Database:** PostgreSQL 16.x
- **ORM:** Entity Framework Core 8.0
- **Authentication:** JWT Bearer tokens (HS256)
- **Security:** BCrypt.Net-Next (password hashing)
- **Documentation:** Swagger/OpenAPI

##  Features

-  User registration with email validation
-  Secure login with JWT token generation
-  Password hashing using BCrypt
-  Clean architecture (Models/DTOs/Services/Controllers separation)
-  PostgreSQL database with EF Core migrations
-  Swagger API documentation
